## 🏆 Featured Project

### 🔬 Synthetic Data & RAG-Assisted Labelling for Multimodal Product Recognition
> 🥇 **Awarded Distinction · Recognised in Top 5 Projects of the Year**

**Architecture:** `Image/Text Input` → `CLIP Embeddings` → `FAISS Retrieval` → `RAG` → `Attribute Prediction`

Built a multimodal product recognition system using Stable Diffusion for synthetic data, CLIP + FAISS for semantic retrieval, and RAG to ground predictions in real catalogue data — reducing hallucinations and improving reliability across head and long-tail product categories.

`Python` `PyTorch` `FAISS` `HuggingFace` `Streamlit` `SQL` `Pandas`
